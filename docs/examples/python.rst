
Python Files
------------

Python files can have comment headers defined using block quotes:

.. code-block:: python

    '''
    This is a valid python docstring

    :author: My Name
    :contact: my_email@gmail.com
    '''

or using a block of start-of-line comments: 

.. code-block:: python

    # This is also a valid comment
    # :author: My Name
    # :contact: my_email@gmail.com
